Overflow Examples

Overflow: visible (default) - This content overflows the box and is still visible outside the boundaries. This is the default behavior.

Overflow: hidden - This content is cut off when it exceeds the box boundaries. Useful for hiding excess content.

Overflow: scroll - Scrollbars are always shown. This content can be scrolled to view the overflow.

Overflow: auto - Scrollbars appear only when needed. This is the most common choice for handling overflow gracefully.
