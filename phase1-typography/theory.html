Color Formats

CSS supports multiple ways to define colors. Choose based on your needs.

1. Hexadecimal (HEX)

Most common format. Uses base-16 notation.

color: #3498db;   /* Full hex: RRGGBB */
color: #fff;      /* Shorthand: RGB (#ffffff) */
color: #3498dbff; /* With alpha: RRGGBBAA */
color: #fff8;     /* Shorthand with alpha */

#00FF00

2. RGB & RGBA

Red, Green, Blue values (0-255).

color: rgb(52, 152, 219);         /* RGB */
color: rgba(52, 152, 219, 0.5);   /* RGBA with 50% opacity */
color: rgb(52 152 219);           /* Modern (no commas) */
color: rgb(52 152 219 / 0.5);     /* Modern with alpha */
  • R, G, B: 0-255 or 0%-100%
  • Alpha: 0-1 (0 = transparent, 1 = opaque)

3. HSL & HSLA

Hue, Saturation, Lightness - more intuitive for humans!

color: hsl(204, 70%, 53%);        /* HSL */
color: hsla(204, 70%, 53%, 0.5);  /* HSLA with opacity */
color: hsl(204deg 70% 53%);       /* Modern syntax */
color: hsl(204 70% 53% / 0.5);    /* Modern with alpha */
Component Range Description
Hue 0-360 Degrees on color wheel (0=Red, 120=Green, 240=Blue)
Saturation 0%-100% 0% = gray, 100% = full color
Lightness 0%-100% 0% = black, 50% = normal, 100% = white
Alpha 0-1 0 = transparent, 1 = opaque
ðŸ’¡ HSL Benefits:
  • Easy to create lighter/darker variations
  • Intuitive color adjustments
  • Great for color schemes
/* Same hue, different lightness */
hsl(204, 70%, 30%)  /* Dark blue */
hsl(204, 70%, 53%)  /* Normal blue */
hsl(204, 70%, 80%)  /* Light blue */

53%
80%

4. Named Colors

CSS has 140+ named colors.

color: red;
color: tomato;
color: cornflowerblue;
color: rebeccapurple;
color: transparent;  /* Fully transparent */
âœ… When to Use Each:
  • HEX: Most common, compact, static colors
  • RGB: Programmatic calculations, transparency
  • HSL: Color schemes, easy variations, best readability
  • Named: Quick prototyping, common colors

Opacity & Transparency

opacity Property

Affects entire element including children.

opacity: 1;    /* Fully opaque (default) */
opacity: 0.5;  /* 50% transparent */
opacity: 0;    /* Fully transparent (invisible) */

Difference: opacity vs alpha

/* opacity affects everything (element + children) */
.box {
  background: red;
  opacity: 0.5;  /* Text inside also 50% transparent */
}

/* Alpha only affects the property */
.box {
  background: rgba(255, 0, 0, 0.5);  /* Only background 50% */
  /* Text remains fully opaque */
}

opacity: 0.5 (everything fades)

Text is also transparent

rgba(255,0,0,0.5) (only background)

Text stays opaque!
ðŸ’¡ Best Practice:

Use alpha channels (rgba/hsla) instead of opacity when you only want to make the background transparent, not the text.

Font Properties

font-family

Defines which font to use. Always provide fallbacks!

font-family: Arial, Helvetica, sans-serif;
font-family: 'Times New Roman', Times, serif;
font-family: 'Courier New', Courier, monospace;
font-family: Georgia, 'Times New Roman', serif;

This is Arial (sans-serif) - clean and modern

This is Times New Roman (serif) - classic and elegant

This is Courier New (monospace) - code-like

This is Georgia (serif) - beautiful for reading

âœ… Font Stack Pattern: 
font-family: 
  'Preferred Font',    /* Your choice */
  'Fallback Font',     /* Similar backup */
  generic-family;      /* serif, sans-serif, monospace */

font-size

font-size: 16px;      /* Pixels (fixed) */
font-size: 1rem;      /* Relative to root (html) */
font-size: 1.2em;     /* Relative to parent */
font-size: 100%;      /* Percentage of parent */
font-size: larger;    /* Keyword */

Common Sizes

html { font-size: 16px; }  /* Base size */

h1 { font-size: 2.5rem; }   /* 40px */
h2 { font-size: 2rem; }     /* 32px */
h3 { font-size: 1.5rem; }   /* 24px */
p  { font-size: 1rem; }     /* 16px */
small { font-size: 0.875rem; } /* 14px */

Heading 2 - 2rem (32px)

Heading 3 - 1.5rem (24px)

Paragraph - 1rem (16px)

Small text - 0.875rem (14px)

font-style

font-style: normal;
font-style: italic;

Normal text - regular style

Italic text - slanted for emphasis

Oblique text - mechanically slanted

Text Properties

text-align

text-align: left;       /* Default */
text-align: right;
text-align: center;
text-align: justify;    /* Stretches lines to fill width */

Text aligned left - This is the default alignment for most text in western languages.

Text aligned center - Perfect for headings and hero sections.

Text aligned right - Useful for RTL languages or special layouts.

Text justified - Stretches the text to fill the entire width, creating clean edges on both sides. This is commonly used in newspapers and magazines for a professional look.

text-decoration

text-decoration: none;           /* Remove underline from links */
text-decoration: underline;
text-decoration: overline;
text-decoration: line-through;   /* Strikethrough */
text-decoration: underline dotted red; /* Style + color */

No decoration - plain text

Underlined text - commonly used for links

Overline text - line above the text

Strikethrough - showing deleted or invalid text

Wavy red underline - like spell check errors

text-transform

text-transform: none;
text-transform: uppercase;    /* ALL CAPS */
text-transform: lowercase;    /* all lowercase */
text-transform: capitalize;   /* First Letter Of Each Word */

none - Original Text As Written

uppercase - all text becomes capitals

LOWERCASE - ALL TEXT BECOMES LOWERCASE

capitalize - first letter of each word

text-indent

text-indent: 2em;    /* Indent first line of paragraph */
text-indent: 50px;

letter-spacing & word-spacing

letter-spacing: 2px;    /* Space between letters */
letter-spacing: 0.1em;
word-spacing: 5px;      /* Space between words */

Normal letter spacing (0)

Increased letter spacing (2px)

Wide letter spacing (5px)

Normal word spacing

Increased word spacing (10px)

text-shadow

/* text-shadow: x-offset y-offset blur color */
text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
text-shadow: 0 0 10px #00ff00;  /* Glow effect */

text-shadow: 
  2px 2px 0 red,

Line Height & Spacing

line-height

Vertical space between lines of text. Critical for readability! 

line-height: 1.5;      /* Unitless (recommended) - multiplier */
line-height: 24px;     /* Fixed pixels */
line-height: 150%;     /* Percentage of font-size */
ðŸ’¡ Best Practices:
  • Use unitless values (1.5, 1.6) - they scale better
  • Body text: 1.5 - 1.8
  • Headings: 1.2 - 1.4
  • Tighter for large text, looser for small text

Line Height Demo

line-height: 1 - Too tight, hard to read with longer text

line-height: 1.5 - Perfect! Comfortable and readable

line-height: 2.5 - Too loose, feels disconnected

Web Fonts

Google Fonts

Easiest way to use custom fonts.

Step 1: Add to HTML <head>

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

Step 2: Use in CSS

body {
  font-family: 'Roboto', sans-serif;
}

@font-face (Custom Fonts)

@font-face {
  font-family: 'MyFont';
  src: url('fonts/myfont.woff2') format('woff2'),
       url('fonts/myfont.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}

body {
  font-family: 'MyFont', sans-serif;
}
âš¡ Performance Tips:
  • Only load weights/styles you actually use
  • Use font-display: swap to prevent invisible text
  • Prefer WOFF2 format (best compression)
  • Limit to 2-3 font families max
