Line Height

Tight line height (1.2). This creates compact text blocks where lines are closer together. Good for headings but can be hard to read in paragraphs.

Normal line height (1.5). This is the recommended line height for body text. It provides good readability with comfortable spacing between lines.

Loose line height (2.0). This creates more breathing room between lines. Can improve readability for some audiences but may feel too spacious.
