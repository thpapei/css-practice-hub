What is Flexbox?

Flexbox (Flexible Box Layout) is a one-dimensional layout system that arranges items in rows or columns. Items can "flex" to fill available space or shrink to prevent overflow.

ğŸ”‘ Key Concept:

Flexbox is one-dimensional, meaning it handles layout in one direction at a time (either row or column).

Flex Container vs Flex Items

.container {
  display: flex;  /* This is now a flex container */
}

/* All DIRECT children of .container are flex items */
  • Flex Container: Parent element with display: flex
  • Flex Items: Direct children of the flex container

Flex Container Properties

Properties you apply to the parent element.

display

.container {
  display: flex;         /* Block-level flex container */
  display: inline-flex;  /* Inline-level flex container */
}

flex-direction

Defines the main axis direction.

.container {
  flex-direction: row;            /* Default: left to right */
  flex-direction: row-reverse;    /* Right to left */
  flex-direction: column;         /* Top to bottom */
  flex-direction: column-reverse; /* Bottom to top */
}

flex-wrap

Controls whether items wrap to new lines.

.container {
  flex-wrap: nowrap;       /* Default: all items on one line */
  flex-wrap: wrap;         /* Items wrap to multiple lines */
  flex-wrap: wrap-reverse; /* Items wrap in reverse order */
}

flex-flow (Shorthand)

.container {
  flex-flow: row wrap;
  /* Same as:
     flex-direction: row;
     flex-wrap: wrap; */
}

justify-content

Aligns items along the main axis.

.container {
  justify-content: flex-start;    /* Default: start of main axis */
  justify-content: flex-end;      /* End of main axis */
  justify-content: center;        /* Center of main axis */
  justify-content: space-between; /* Equal space between items */
  justify-content: space-around;  /* Equal space around items */
  justify-content: space-evenly;  /* Equal space including edges */
}

align-items

Aligns items along the cross axis.

.container {
  align-items: stretch;      /* Default: fill container height */
  align-items: flex-start;   /* Align to start of cross axis */
  align-items: flex-end;     /* Align to end of cross axis */
  align-items: center;       /* Center on cross axis */
  align-items: baseline;     /* Align text baselines */
}

align-content

Aligns multiple lines on cross axis (when wrapping). 

.container {
  flex-wrap: wrap;  /* Required for this to work */
  align-content: flex-start;
  align-content: flex-end;
  align-content: center;
  align-content: space-between;
  align-content: space-around;
  align-content: stretch;  /* Default */
}

gap

Space between flex items (modern browsers).

.container {
  gap: 20px;          /* Equal gap between all items */
  gap: 20px 10px;     /* Row gap | Column gap */
  row-gap: 20px;      /* Only vertical gap */
  column-gap: 10px;   /* Only horizontal gap */
}

Flex Item Properties

Properties you apply to the children (flex items).

flex-grow

How much the item should grow relative to others.

.item {
  flex-grow: 0;  /* Default: don't grow */
  flex-grow: 1;  /* Grow to fill available space */
  flex-grow: 2;  /* Grow twice as much as items with flex-grow: 1 */
}

Example

/* Container has 300px extra space */
.item1 { flex-grow: 1; }  /* Gets 100px */
.item2 { flex-grow: 2; }  /* Gets 200px */
flex-grow: 1
flex-grow: 2
(twice as wide)

flex-shrink

How much the item should shrink when space is tight.

.item {
  flex-shrink: 1;  /* Default: can shrink */
  flex-shrink: 0;  /* Don't shrink */
  flex-shrink: 2;  /* Shrink twice as much */
}

flex-basis

Initial size of item before growing/shrinking.

.item {
  flex-basis: auto;   /* Default: based on content */
  flex-basis: 200px;  /* Start at 200px */
  flex-basis: 50%;    /* 50% of container */
  flex-basis: 0;      /* Common for equal-width items */
}

flex (Shorthand)

Combines flex-grow, flex-shrink, and flex-basis. 

.item {
  flex: 1;        /* flex-grow: 1, flex-shrink: 1, flex-basis: 0 */
  flex: 0 0 auto; /* Don't grow or shrink */
  flex: 1 0 200px; /* grow: 1, shrink: 0, basis: 200px */
}
ğŸ’¡ Common Patterns: 
/* Equal width items that grow */
.item { flex: 1; }

/* Fixed width item */
.item { flex: 0 0 200px; }

/* Grow but don't shrink */
.item { flex: 1 0 auto; }

align-self

Override align-items for individual item.

.item {
  align-self: auto;       /* Default: use container's align-items */
  align-self: flex-start;
  align-self: flex-end;
  align-self: center;
  align-self: stretch;
  align-self: baseline;
}

order

Change visual order of items (default is 0).

.item1 { order: 2; }  /* Appears third */
.item2 { order: 1; }  /* Appears second */
.item3 { order: 0; }  /* Appears first (default) */

Common Flexbox Patterns

Center Everything

.container {
  display: flex;
  justify-content: center;  /* Center horizontally */
  align-items: center;      /* Center vertically */
  min-height: 100vh;        /* Full viewport height */
}

Navbar

.navbar {
  display: flex;
  justify-content: space-between;  /* Logo left, menu right */
  align-items: center;
  padding: 1rem 2rem;
}

.nav-menu {
  display: flex;
  gap: 2rem;  /* Space between menu items */
}

Card Grid

.card-container {
  display: flex;
  flex-wrap: wrap;
  gap: 2rem;
}

.card {
  flex: 1 1 300px;  /* Grow, shrink, min 300px */
  /* Cards will wrap to new line when < 300px */
}

Sticky Footer

body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

main {
  flex: 1;  /* Takes all available space */
}

footer {
  /* Stays at bottom */
}

Equal Height Columns

.container {
  display: flex;
}

.column {
  flex: 1;  /* All columns equal width */
  /* All columns automatically same height! */
}
